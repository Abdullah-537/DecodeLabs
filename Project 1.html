Core Features

Built on the Right Foundations

Every pixel placed with purpose. Every element semantic by default.

📐

Semantic HTML5

Meaningful landmark elements — header, nav, main, article, footer — that screen readers and search engines understand natively.

📱

Mobile-First CSS

Single-column base styles expand progressively at 768px (tablet) and 1024px (desktop) using CSS Grid and Flexbox without any framework.

⚡

Vanilla JavaScript

Scroll-reveal, hamburger menu, theme toggle, and sticky nav — all done with clean, readable vanilla JS. No jQuery, no libraries.

🎨

2025 Design System

Mocha Mousse, Ethereal Blue, and Moonlit Grey palette with Cormorant Garamond display and DM Sans body — exactly as specified in the brief.

♿

Accessibility First

WCAG-compliant with proper ARIA labels, keyboard navigation, focus management, and sufficient color contrast throughout.

🌙

Dark Mode Toggle

One-click dark/light theme switch using CSS custom properties — zero JavaScript DOM queries needed for the color system.
